# HHKBt

Happy Hacking Keyboard Bluetooth

## Features

* Send HID keyboard status via RN41/42 module
* 5V USB supply
* 10 hours battery life with 2700mAh eneloop mobile booster

## Requirement

### Library

https://github.com/felis/USB_Host_Shield_2.0
